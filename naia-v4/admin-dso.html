FR-DSO · un mandat par centrale, généré au sign-up. Le template est résolu depuis pays + région (closed enum, pas de texte libre · FR-DSO-4). Les templates sont versionnés ; chaque mandat émis garde la version au moment du signing (FR-DSO-3). Une zone sans template configuré bloque le sign-up avec message clair. Sur signature, l'événement plant.dso_mandate_signed déclenche la connexion DSO via le module Insight (FR-DSO-5).
