## What stays here

This folder keeps a few design artifacts that are hosted on GitHub Pages:

- `cabinet-panels.html` — interactive SVG panel drawings ([live preview](https://hydro-software.github.io/Designs-wireframes/conference-game/cabinet-panels.html))
- `reel/` — built Remotion conference reel viewer ([live preview](https://hydro-software.github.io/Designs-wireframes/conference-game/reel/))
- `naia-arcade-panel-*.png / .pdf` — vinyl decal artwork for the arcade cabinet lightbox
- `20260420_*.jpg` — reference photos of the Acer monitor used in the cabinet build

These will migrate to `hydro-software/game`'s own GitHub Pages once the game repo is set up for it.
